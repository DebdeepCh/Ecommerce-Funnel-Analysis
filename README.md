# 📊 E-commerce Funnel Analysis 

## 📌 Overview
This project analyzes customer behavior in an e-commerce dataset using MySQL. Since the dataset does not contain explicit funnel events, a business-driven funnel is created using purchase behavior.

---

## 🎯 Objectives
- Build a custom customer funnel from transactional data  
- Analyze conversion rates and drop-offs  
- Perform cohort and retention analysis  
- Identify churn patterns and high-value customers  

---

## 🧾 Dataset
Includes:
- Customer ID, Purchase Date  
- Product Category, Price, Quantity  
- Payment Method, Returns  
- Churn status, Gender  

---

## 🔄 Funnel Stages
- Visitors → Unique customers  
- Interested → At least 1 purchase  
- Engaged → 2+ purchases  
- Loyal → 5+ purchases  
- Retained → Non-churned customers  

---

## ⚙️ Tools Used
- MySQL (CTEs, Window Functions, Aggregations)  

---

## 📊 Key Insights
- Most customers drop off after first purchase  
- Small customer segment drives majority of revenue  
- Retention decreases over time  
- Product categories show varying return rates  

---
